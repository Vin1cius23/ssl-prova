## Dificuldades

Tive dificuldades na organização dos arquivos nas pastas corretas e no uso do Git para adicionar e enviar os arquivos. Também houve dificuldade na configuração do SSH para utilizar o comando SCP, pois o serviço não estava ativo inicialmente.

## Conclusão

A atividade permitiu compreender na prática o funcionamento de certificados digitais, geração de chaves criptográficas e utilização de protocolos seguros. Foi possível entender como ocorre a comunicação segura na web e a importância dos certificados para garantir a autenticidade e segurança das conexões.
